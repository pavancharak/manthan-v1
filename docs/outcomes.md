\# Decision Outcomes



Manthan separates:



\- status → evaluation state

\- outcome → decision result



\---



\## Status



\- INVALID → bad input

\- INCOMPLETE → missing data

\- DECIDED → evaluation complete



\---



\## Outcome (only when DECIDED)



\- ALLOW → safe to proceed

\- BLOCK → must not proceed

\- ESCALATE → requires external decision



\---



\## Important



ESCALATE is a valid decision.



INCOMPLETE is not a decision.
