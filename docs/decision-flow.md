\# Decision FlowManthan evaluates input in strict order:1\. Validate input2\. Check completeness3\. Evaluate rules4\. Return result\---\## Flowvalidate → completeness → rules → outcome\---\## Outcomes\### INVALIDInput does not match schema\### INCOMPLETERequired fields are missing\### DECIDEDRules evaluated successfully\- ALLOW → proceed\- BLOCK → deny\- ESCALATE → defer decision
